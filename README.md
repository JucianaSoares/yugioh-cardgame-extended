# 🃏 Yu-Gi-Oh! Card Game - Edition: Survival Duel

[link do projeto](https://jucianasoares.github.io/yugioh-cardgame-extended/)

Este é um projeto desenvolvido com base no desafio de código da **DIO (Digital Innovation One)**, totalmente customizado, expandido e otimizado com mecânicas avançadas de jogabilidade, persistência de dados e efeitos visuais dinâmicos.

## 🚀 Novas Funcionalidades (Além do Escopo Original)

Para transformar o projeto base em um jogo mais robusto e divertido, implementei as seguintes mecânicas utilizando JavaScript Vanilla e CSS3:

* **✨ Sistema de Raridades Dinâmico:** As cartas agora possuem classificações de raridade (`common`, `rare`, `legendary`). Cartas raras e lendárias ganham bordas e auras de brilho pulsante personalizados via CSS no campo de batalha.
* **🌋 Tabuleiro Dinâmico (Ambientação Responsiva):** O cenário do duelo reage ao tipo de monstro invocado! Invocar um **Dragão** transforma a arena em um ambiente vulcânico, enquanto **Magos** ativam uma aura arcana roxo-neon e o **Exodia** desencadeia um brilho dourado místico.
* **💾 Persistência de Dados com `localStorage`:** O jogo possui um placar global de Vitórias e Derrotas que fica salvo diretamente no navegador do usuário. Mesmo se a página for atualizada ou fechada, o histórico não é perdido.
* **💀 Modo Sobrevivência (Survival Mode):** Uma mecânica de maratona infinita! O dano recebido pelo jogador acumula para os próximos turnos. Ao derrotar a IA, um novo oponente com 8.000 LP é gerado, desafiando o jogador a ver quantos duelos ele consegue vencer em sequência antes de ter seus pontos zerados.
* **🌟 Vitória Instantânea (Exodia):** Fiel ao anime, invocar a carta do Exodia causa dano massivo instantâneo, derrotando o oponente atual na hora e avançando o jogador no Modo Sobrevivência.

## 📱 Design 100% Responsivo
O layout do jogo foi planejado e reconstruído com foco na **responsividade**. Ele se adapta perfeitamente tanto a telas de computadores quanto a dispositivos móveis (smartphones e tablets), reorganizando o tabuleiro, o painel de detalhes e as cartas da mão para garantir uma jogabilidade confortável e fluida em qualquer resolução.

## 🛠️ Tecnologias Utilizadas

* HTML5 (Estrutura do tabuleiro e painel de detalhes)
* CSS3 (Media Queries para responsividade, efeitos `box-shadow`, animações `@keyframes` e gradientes radiais)
* JavaScript (Lógica assíncrona, manipulação segura do DOM no ciclo de inicialização e persistência de dados)

## 🔮 Próximos Passos & Visão de Futuro (Roadmap)
O meu grande desejo e meta para este projeto é transformá-lo em um jogo **Multiplayer Online em Tempo Real**. 

Assim que eu dominar a base de desenvolvimento backend com **Node.js** e comunicação bidirecional via **WebSockets (Socket.io)**, pretendo implementar uma arquitetura de servidor para gerenciar salas de jogos (lobbies), permitindo que dois jogadores reais se enfrentem através da internet com total sincronização de turnos e segurança na lógica de combate.

---
*Projeto totalmente codificado e testado em ambiente mobile utilizando o editor Acode integrado ao Termux.*

# 🃏 Yu-Gi-Oh! Card Game - Edition: Survival Duel

This project was developed based on a coding challenge from **DIO (Digital Innovation One)**, fully customized, expanded, and optimized with advanced gameplay mechanics, data persistence, and dynamic visual effects.

## 🚀 New Features (Beyond the Original Scope)

To turn the base project into a more robust and fun game, I implemented the following mechanics using Vanilla JavaScript and CSS3:

* **✨ Dynamic Rarity System:** Cards now have rarity classifications (`common`, `rare`, `legendary`). Rare and legendary cards get customized pulsing neon borders and glows via CSS when on the battlefield.
* **🌋 Dynamic Board (Responsive Theme):** The dueling arena reacts to the type of monster summoned! Summoning a **Dragon** transforms the field into a volcanic environment, while **Magicians** activate a purple-neon arcane aura, and **Exodia** unleashes a mystical golden glow.
* **💾 Data Persistence with `localStorage`:** The game features a global Wins and Losses scoreboard saved directly in the user's browser. Even if the page is refreshed or closed, your match history is preserved.
* **💀 Survival Mode:** An endless marathon mechanic! Damage taken by the player carries over to the next rounds. Upon defeating the AI, a new opponent with 8,000 LP is generated, challenging the player to see how many duels they can win in a row before their points hit zero.
* **🌟 Instant Victory (Exodia):** True to the anime, summoning the Exodia card deals massive instant damage, immediately defeating the current opponent and advancing the player in Survival Mode.

## 📱 100% Responsive Design
The game layout was planned and rebuilt with a focus on **responsiveness**. It adapts perfectly to both computer screens and mobile devices (smartphones and tablets), rearranging the board, detail panel, and hand cards to ensure comfortable and smooth gameplay on any resolution.

## 🛠️ Technologies Used

* HTML5 (Board structure and details panel)
* CSS3 (Media Queries for responsiveness, `box-shadow` effects, `@keyframes` animations, and radial gradients)
* JavaScript (Asynchronous logic, safe DOM manipulation within the initialization lifecycle, and data persistence)

## 🔮 Next Steps & Roadmap
My ultimate goal for this project is to transform it into a **Real-Time Online Multiplayer** game.

As soon as I master backend development fundamentals with **Node.js** and bidirectional communication via **WebSockets (Socket.io)**, I plan to implement a server architecture to manage game lobbies, allowing two real players to face each other over the internet with fully synchronized turns and secure combat logic.
